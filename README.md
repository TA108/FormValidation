# Simple Form Validation with jQuery

This project is a basic HTML form with client-side validation using jQuery.  
It validates user input for Email, Phone Number, Password, and Confirm Password fields.

---

## Features

- Checks if Email, Phone Number, and Password fields are filled.
- Validates Email format using regex.
- Validates that Phone Number contains only numbers.
- Checks if Password and Confirm Password match.
- Displays error messages dynamically.
- Shows success message upon valid submission.

---

## Files

- `index.html` — The main HTML page containing the form.
- `style.css` — CSS file for styling the form and error messages.
- `script.js` — JavaScript file with jQuery-based validation logic.

---

## How to Use

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Fill out the form fields and click "Register".
4. Error messages will appear if inputs are invalid.
5. A success message will display if all inputs are valid.

---

## Dependencies

- [jQuery](https://jquery.com/) (loaded via CDN in `index.html`)

---

## Notes

- This is a front-end only validation example. For production use, always validate input on the server side as well.
- Customize styles in `style.css` as needed.
